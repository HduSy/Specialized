# Content

`WebContainer` 基于浏览器运行 Nodejs Application 和操作系统命令的运行时。

## 与云虚拟机比较

- 完全运行在本地
- （增效）极致用户体验，速度快，几乎无延迟；可离线
- 浏览器安全机制保证
- （降本）开源免费，无需支付云主机费用
