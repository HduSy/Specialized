### 一、绝对定位居中
- `transform`  手段
```scss
width: 100px;
height: 100px;
position: absolute;
left: 50%;
top: 50%;
transfrom: translate(-50%, -50%);
```
- `calc` 手段
```scss
width: 100px;
height: 100px;
position: absolute;
left: calc(50% - 50px);
top: calc(50% - 50px);
```
