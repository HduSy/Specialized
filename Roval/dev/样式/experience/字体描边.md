## 伪元素 + text-stroke 描边

代码：

```scss
.value {  
  font-family: xx-bin;  
  font-size: 24px;  
  font-weight: 700;  
  line-height: 22px;  
  color: #FFF1C5;  
  -webkit-text-stroke: 3px #823700;  
  &::before {  
    content: attr(data-v);  
    position: absolute;  
    top: 2.5px;  
    background-color: #FFF1C5;  
    background-clip: text;  
    -webkit-background-clip: text;  
    -moz-background-clip: text;  
    -webkit-text-stroke-width: 0;  
    color: transparent;  
  }  
}
```
