## 相关 API

### runCallbacksOnInit

如果你的初始化 slide 不是第一个 (例 initialSlide:2) 或者设置了 loop: true，那么初始化时会触发一次 [Transition/SlideChange] [Start/End] 回调函数。  
如果不想触发，可将此参数设置为 false。

### 双向控制

可能会导致重复请求
